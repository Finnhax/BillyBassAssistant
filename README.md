# Billy Bass Assistant
A billy the bass mod to use as a voice assistant for home assistant.

## Hardware:

| Part                 | Hardware              |
| -------------------- | --------------------- |
| Big Mouth Billy Bass | The fish              |
| Motor Driver         | L298N                 |
| Amplifier            | MAX98357              |
| Mic                  | INMP441               |
| Board                | ESP32-S3-devkitc-1    |
